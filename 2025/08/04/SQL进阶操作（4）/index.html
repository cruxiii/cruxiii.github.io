DROP

通过DROP语句，可以轻松删除索引、表、数据库。

DROP INDEX

DROP INDEX 用于删除表中索引

语法

DROP TABLE

DROP TABLE 用于删除表。

DROP DATABASE

DROP DATABASE 用于删除数据库，及其中的所有表、视图等。

ALTER TABLE

ALTER TABLE 用于在已有的表中，添加、删除或者修改列。

语法

  • 在表中添加列
  • 在表中删除列
  • 改变表中数据类型
